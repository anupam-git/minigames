# Asteroids
A qml implemetation of classic Asteroids game

## How to Run
`qmlscene asteroids.qml`

## How to play
  - Press left and right arrow keys to rotate spaceship
  - Press up and down arrow keys to move spaceship
  - Press Space key to shoot bullets
